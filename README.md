## Overview

Determines if a number is a Happy Number.

A Happy Number is a number which eventually reaches 1 when replaced by the sum of the square of each digit.

## Function

### `happy_num(x: int) -> str`

- **Description**: Checks whether a given number is a Happy Number.
- **Parameters**:
  - `x`: The number to check for happiness (type: int).
- **Returns**:
  - A message indicating whether the number is a Happy Number or not (type: str).

## Usage

```python
# Example Usage
print(happy_num(7))   # Output: 7 --> Happy Number :)
print(happy_num(13))  # Output: 13 --> Happy Number :)
print(happy_num(45))  # Output: 45 --> is Not a Happy Number :(
print(happy_num(99))  # Output: 99 --> is Not a Happy Number :(
```
